## 📌 **FUNDAMENTAÇÃO CONCEITUAL**

Este squad foi construído com base nas últimas pesquisas sobre a evolução do marketing na era agentica:

- **A Evolução das Estratégias**:
  - **SEO** (tráfego via links) → **AEO** (respostas diretas) → **GEO** (citações em respostas geradas) → **ACO** (transações por agentes).
- **Os 3 Tipos de Agentes no Comércio**:
  - **Onsite Agents**: Agentes no próprio site (ex: assistente da Artificiall)
  - **Operator Agents**: Agentes que navegam pela web pelo usuário (ex: ChatGPT Operator)
  - **Protocol Agents**: Agentes que transacionam via API (ACP/UCP).
- **A Nova Moeda: "Citation Share"**: O sucesso não é mais medido por backlinks, mas por quantas vezes a marca é citada nas respostas das IAs.
- **Os 7 Pilares do ACO**:
  1. Dados calculáveis (não apenas descritivos)
  2. Schema como "cola" para agentes
  3. Motor de confiança (políticas claras, garantias)
  4. Acionabilidade (checkout via API)
  5. Interoperabilidade (padrões ACP/UCP)
  6. Testes com agentes
  7. Métricas de "prateleira invisível".
